**项目简介**：MoonBit 生态通用图算法工具箱。提供泛型有向图 `Graph[N, E]` 及 **30+ 种**经典图论与基础算法的原生实现，覆盖遍历、最短路、最小生成树、网络流、匹配、连通性、着色、中心性与高级数据结构。配套完整单元测试（188 项，全部通过，`moon check --deny-warn` / `moon test --deny-warn` 均为 0 警告 0 错误）。

**与已有包的差异化**：mooncakes.io 上现有图算法包（如 MoonGraph、moonpath）主要覆盖遍历 / 最短路 / 路径规划。本项目独有的 **最小生成树、网络流、全源最短路、二分图匹配、匈牙利指派、欧拉路、割点桥、强连通分量、2-SAT、传递闭包、LCA、图着色、中心性度量** 等核心能力，形成互补而非重复。

**参考说明**：本项目为**原创实现**，非移植。所有算法均为经典图论算法的独立 MoonBit 原生实现，参考 CLRS（算法导论）等标准教材与 Wikipedia 技术文档，未直接复制任何第三方开源库代码。

**与原项目/教材的差异**：使用 MoonBit 泛型系统 `Graph[N, E]` 实现类型安全的图结构；统一返回 `Option[T]` 避免 panic；全部算法配套完整单元测试，覆盖边界条件（平行边、负权、空图、单点、不可达、孤立点、非连通结构等）；API 适配 MoonBit 包管理规范，已发布至 mooncakes.io。
